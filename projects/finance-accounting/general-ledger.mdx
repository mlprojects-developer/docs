# General Ledger

General ledger and chart of accounts.

```mermaid
graph TD
    A[Transaction] --> B[Journal Entry]
    B --> C[Post]
    C --> D[Ledger]
    D --> E[Reports]
```

## Features

- Chart of accounts
- Journal entries
- Multi-currency support
- Intercompany transactions
- Account reconciliation
- Balance verification
- Period closing
- Account analysis
