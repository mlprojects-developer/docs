# Scheduled Reports

Automate report generation and distribution on a schedule.

```mermaid
stateDiagram-v2
    [*] --> Scheduled
    Scheduled --> Waiting
    Waiting --> Trigger_Time
    Trigger_Time --> Generate
    Generate --> Format
    Format --> Distribute
    Distribute --> Archive
    Archive --> [*]
```

## Scheduling Options

- Daily
- Weekly
- Monthly
- Quarterly
- Custom intervals
- Business days only
- Holiday awareness

- File storage
- Webhook delivery
- Slack messages
- API endpoints
