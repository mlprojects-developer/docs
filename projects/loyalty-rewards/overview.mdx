# Loyalty & Rewards

Loyalty program and customer rewards management.

## System Context

```mermaid
C4Context
    title Loyalty & Rewards System Context
    Person(member, "Program Member", "Earns and redeems points")
    System(loyalty, "Loyalty Platform", "Points and rewards management")

    System_Ext(crm, "CRM System", "Customer data")
    System_Ext(email, "Email Service", "Engagement campaigns")

    Rel(ecom, loyalty, "Sends purchase data")
```

## System Containers

```mermaid
C4Container
    title Loyalty Platform Containers
    System_Boundary(loyalty, "Loyalty Platform") {

        Container(engine, "Points Engine", "Java", "Calculates and awards points")
        ContainerDb(db, "Database", "PostgreSQL", "Member and points data")
        Container(cache, "Cache", "Redis", "Member rewards cache")
    }
    System_Ext(crm, "CRM System", "Customer profiles")
    
    Rel(web, api, "REST")
    Rel(api, engine, "RPC")
    Rel(engine, db, "Reads/Writes")
    Rel(api, cache, "Reads/Writes")
    Rel(api, crm, "API calls")
```
