# Knowledge Management

Enterprise knowledge management system.

## System Context

```mermaid
C4Context
    title Knowledge Management System Context
    Person(user, "User", "Searches and reads content")
    Person(author, "Author", "Creates and publishes content")
    System(km, "Knowledge Platform", "Content management and discovery")
    System_Ext(ldap, "Directory Service", "User authentication")
    System_Ext(email, "Email Service", "Notifications")
    
    Rel(user, km, "Searches")
    Rel(author, km, "Publishes")
    Rel(km, ldap, "Authenticates")

```

## System Containers

```mermaid
C4Container

    System_Boundary(km, "Knowledge Platform") {
        Container(web, "Web Portal", "React", "Browse and search")

        Container(search, "Search Index", "Elasticsearch", "Full-text search")

    System_Ext(auth, "Auth Service", "LDAP/OAuth")
    
    Rel(web, api, "REST")
    Rel(editor, api, "REST")
    Rel(api, db, "Reads/Writes")
    Rel(api, search, "Indexes")
    Rel(api, auth, "Validates")
```
