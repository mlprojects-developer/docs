# E-commerce

Complete e-commerce platform for online selling and product management.

## System Context

```mermaid
C4Context
    title E-Commerce System Context
    Person(customer, "Customer", "Browses and purchases products")
    System(ecommerce, "E-Commerce Platform", "Complete online store solution")
    System_Ext(payment, "Payment Provider", "Processes payments")
    System_Ext(shipping, "Shipping Provider", "Manages deliveries")
    System_Ext(email, "Email Service", "Sends notifications")
    
    Rel(customer, ecommerce, "Uses")
    Rel(ecommerce, payment, "Processes payments")
    Rel(ecommerce, shipping, "Requests deliveries")
    Rel(ecommerce, email, "Sends notifications")
```

## System Containers

```mermaid
C4Container
    title E-Commerce System Containers
    System_Boundary(ecom, "E-Commerce Platform") {
        Container(web, "Web Application", "React", "Customer-facing storefront")
        Container(api, "API Server", "Node.js", "Business logic and data")
        Container(admin, "Admin Portal", "React", "Store management")
        ContainerDb(db, "Database", "PostgreSQL", "Product and order data")
        Container(cache, "Cache", "Redis", "Session and product cache")
    }
    System_Ext(payment, "Payment Gateway", "Stripe/PayPal")
    
    Rel(web, api, "REST/GraphQL")
    Rel(admin, api, "REST/GraphQL")
    Rel(api, db, "Reads/Writes")
    Rel(api, cache, "Reads/Writes")
    Rel(api, payment, "Processes")
```
