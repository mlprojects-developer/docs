# Inventory Audit

Tools for conducting physical inventory counts and reconciliation.

```mermaid
stateDiagram-v2
    [*] --> Planning
    Planning --> Counting
    Counting --> Data_Entry
    Data_Entry --> Verification
    Verification --> Analysis
    Analysis --> Reconciliation
    Reconciliation --> [*]
```

## Audit Features

- Cycle counting
- Full physical inventory
- Spot checks
- Bin counting
- Mobile counting apps
- Count verification
- Audit reports
- Discrepancy tracking

## Reconciliation

- System vs actual comparison
- Variance analysis
- Root cause analysis
- Adjustment posting
- Approval workflow
- Historical tracking
- Trend analysis

## Process

- Plan audit
- Create count batches
- Assign counters
- Mobile data entry
- Verify counts
- Analyze discrepancies
- Post adjustments
- Generate reports
